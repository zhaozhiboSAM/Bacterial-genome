This a shell script for rapid detection and annotation of Tn DNA insertion in bacteria, based on Illumina sequences.

*******
If you use Linux systems, please copy the 'BTTDA.linux.bin_files.v1/bin/*' to replace the './BTTDA.for.Mac.v1/bin/*'. 
Then run as described in 'BTTDA.linux.bin_files.v1/readme.txt'.

Please download the linux version 'gtfToGenePred' from https://hgdownload.cse.ucsc.edu/admin/exe/macOSX.x86_64/, download linux version blastn 2.2.28+ from https://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/





Note: 
(1) The .sh file permission can be given:
chmod 755 ./run_*.sh ./bin/*
(2) The script has been tested on Ubuntu 20. 
(3) The pipeline script written by Zhibo Zhao at Guizhou University [e-mail: zbzhao@gzu.edu.cn].
